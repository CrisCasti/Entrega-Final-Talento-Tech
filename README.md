# Portfolio Freelancer - Pre-entrega Talento Tech

Este es el proyecto desarrollado para la pre-entrega del curso **Talento Tech**. Se trata de una página web personal para un Diseñador Freelancer, enfocada en la presentación de servicios y captura de clientes mediante un formulario de contacto funcional.

## Demo
Podés ver el sitio en vivo aquí: https://criscasti.github.io/Pre-entrega-talentotech/

##  Tecnologías utilizadas
*   **HTML5:** Estructura semántica completa.
*   **CSS3:** Diseño personalizado utilizando:
    *   **Flexbox:** Para la navegación y alineación de elementos.
    *   **CSS Grid:** Para la disposición de la sección de servicios y el formulario.
    *   **Variables CSS:** Para una gestión eficiente de la paleta de colores.
    *   **Diseño Responsivo:** Adaptado a dispositivos móviles y escritorio (Mobile First).
*   **Formspree:** Integración para la gestión de envíos del formulario de contacto sin necesidad de backend.
*   **Fuentes de Google:** Uso de la tipografía "Krub".

##  Requerimientos Cumplidos
1. **Estructura HTML:** Uso de etiquetas semánticas (`header`, `nav`, `main`, `section`, `footer`).
2. **Formulario de Contacto:** Vinculado con Formspree para recibir consultas reales.
3. **Estilos CSS:** Implementación de normalización de estilos y diseño moderno.
4. **Layout:** Uso de Flexbox y Grid para un diseño flexible.
5. **Multimedia:** Iconos en formato SVG para una carga rápida y escalable.
